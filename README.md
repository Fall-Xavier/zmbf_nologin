### Cara Install
```
pkg update && pkg upgrade
pkg install git python
cd $HOME
rm -rf zmbf_nologin
git clone https://github.com/Fall-Xavier/zmbf_nologin
cd zmbf_nologin
python -m pip install -r requirements.txt
python build.py
python run.py
```
